## v0.2.0 Changelog

Now you can open your saved queries using `CTRL+E CTRL+O`!

### Improvements
* Added opening support for saved queries (Issue #12)
```javascript
// new hotkeys setting
{
    "keys": ["ctrl+e", "ctrl+o"], 
    "command": "st_list_queries", 
    "args": {
        "mode" : "open"
    }
}
```
